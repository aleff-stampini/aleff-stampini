# Aleff Stampini

**`Desenvolvedor Java`**

Desenvolvedor Back-end Java com mais de 4 anos de experiência em desenvolvimento de APIs REST e microserviços utilizando Spring Boot.

Experiência sólida em Java, SQL, AWS, Docker e Apache Kafka, atuando em sistemas escaláveis, distribuídos e de alta performance.

Atuação em projetos de meios de pagamento, telecomunicações e produtos de crédito, com foco em integração entre sistemas, arquitetura backend e boas práticas de desenvolvimento.

Principais competências:
- Desenvolvimento de APIs REST
- Arquitetura de microserviços
- Integração de sistemas
- Sistemas escaláveis e de alta performance
- Cloud (AWS) e containers (Docker)

Atualmente atuo como Desenvolvedor Java Pleno, trabalhando com sistemas financeiros e soluções backend críticas.
